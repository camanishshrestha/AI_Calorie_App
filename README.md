# AI Calorie Tracker

A Streamlit app to track calories, calculate macros, and predict food from photos using AI.

## Features
- Manual food logging
- Daily calorie & macro tracking
- AI meal photo recognition
- Auto logging calories from AI predictions
- Progress bar and macro pie chart visualization
